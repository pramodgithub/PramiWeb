## Demo project - React+Redux , Java - SpringBoot, SQL

## Getting started

- Recommended `node js 16.x`, `Java 8+`, `MySQL` and `npm 6+`. (suggestion v16.15.0)
- Install dependencies: `npm install` / `yarn install`
- Start the project: `npm run start` / `yarn start`
